How to Install and Use VS Code's Live Server Extension Setup

Install | Use | Video

Installation Instructions

  1. In Visual Studio Code, click the Extensions icon in the left column toolbar.
  2. In the search bar that appears, type Server.
  3. Select the Live Server Extension. It should be the top choice.
  4. In the page for Live Server that appears, click the green Install button.

Usage Instructions

  • Visual Studio Code must be open to a folder to use Live Server.
  • In VS Code's Explorer, right click (Windows) or Control+click (macOS) on HTML file. From the context menu that appears, select Open with Liver Server.
